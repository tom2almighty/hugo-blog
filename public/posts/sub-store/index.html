Docker部署Sub-Store后端
使用Docker部署Sub-Store，全平台管理订阅。
2024-04-08 10:58 folder 上网   label Docker   sub-store   订阅转换   schedule 1 min 7 s.

前言

Sub-Store 是一个适用于 Loon 、 Surge 和 Quantumult X 的高级订阅管理工具。完全本地解析，无订阅泄露的风险。相关地址及文档见下方：

  • Github 地址
  • 使用教程
  • 自建教程

部署

mkdir -p /opt/sub-store && cd /opt/sub-store
cat > /opt/sub-store/docker-compose.yml <<EOF
version: "3.8"
services:
  sub-store:
    image: xream/sub-store:latest
    container_name: sub-store
    restart: always
    volumes:
      - /opt/sub-store:/opt/app/data
    environment:
      - SUB_STORE_FRONTEND_BACKEND_PATH=/xxx  # 20位随机字符串  
      - SUB_STORE_CRON=1 0 * * * 
    ports:
      - 3001:3001
    stdin_open: true
    tty: true
EOF
docker compose up -d

随机字符串生成可以使用下面的网站：

  • IT 工具箱
  • 1Password 密码生成

反向代理

使用 Nginx 或者其他工具反向代理，端口为 3001，代理完成后访问的地址为：

https://你的域名?api=https://你的域名/刚刚生成20位随机字符串

更新

cd /opt/sub-store
docker compose down
docker compose pull
docker compose up -d
